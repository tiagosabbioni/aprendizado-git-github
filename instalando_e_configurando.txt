Podemos verificar as configurações do Git utilizando o comando "git config", e podemos também definir um nome de usuário e um e-mail utilizando os comandos "git config --global user.name (nome)" e "git config --global user.email (endereço)", sendo "--global" um parâmetro que define essa configuração para todos os projetos. Além disso, podemos utilizar o Git através de um terminal baseado em Unix como os terminais de distribuições Linux ou MacOS. Para inicializar um projeto git, basta utilizarmos o comando "git init".
