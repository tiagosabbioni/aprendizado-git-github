Após criar uma conta no Github, podemos clonar repositórios e realizar pushes e pulls para repositórios remotos. Para clonar podemos usar o comando 'git clone (url)', já para configurar um repositório remoto utilizamos o comando 'git remote add origin (url)'. Para fazer um pull utilizamos 'git pull origin (nome-da-branch)' e para fazer um push utilizamos o comando 'git push origin (nome-da-branch)'.
