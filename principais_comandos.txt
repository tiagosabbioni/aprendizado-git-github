Alguns dos principais comando git são o 'git init' que nos permite inicializar um projeto git, o 'git status' que nos permite ver quais arquivos sofreram alterações, o 'git diff' que nos permite ver o que foi alterado em cada arquivo, o 'git add' que nos permite acrescentar arquivos a serem "commitados" para a HEAD, o 'git commit -m' que faz um commit já com uma mensagem, o 'git reset --hard (HEAD)' que nos permite visitar uma HEAD anterior do projeto e o 'git checkout' que restaura os arquivos para suas últimas versões sem alteração.
