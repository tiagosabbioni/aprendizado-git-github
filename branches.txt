Através da criação de branches podemos ramificar o projeto de forma que mais pessoas possam trabalhar nele sem modificar os arquivos principais e então, ao concluir as alterações, unir a nova branch à branch master. Para criar uma nova branch podemos usar o comando 'git branch (nome)' ou então utilizando o shorthand 'git checkout -b (nome)'. Para alternar entre branches, basta utilizarmos o comando 'git checkout (nome)'. Ao concluir os trabalhos em uma branch, basta executar o comando 'git merge (nome)' na branch master.
